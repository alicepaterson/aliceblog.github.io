Week 1

This week we began the course and learnt some basic tags in html, including paragraphs, lists and fonts. I learnt the basic set up for a html document, added a header, menu bar and a contact me button to the page, although these are static and non-responsive at the moment. I also learnt how to change the background using links with different images, which I am uploading to imgur.com. I can also now add borders to images.

Week 2

This week we began using css to format our webpages. I have changed the menu bar background to be transparent using rgba colours. The fonts on this page are linked to googlefonts, and I have used classes/ids to specify the style of individual elements on my page. I have also been playing with the font-weight to make the text more readable on this page

Week 3

This week we learnt what GitHub is, how to create a repository, branches, commits, and how to pull requests. I have uploaded the local files for my site to GitHub and am now using that to track the changes I am making. I also published this page using GitHub pages!As part of my homework I have looked at how to make my page more responsive, and I have added hover effects to the menu links so that they become larger when you hover over them. I have also done some reading about Javascript and have added a 'back to top' button to my blog page, which fades into view when you scroll to the bottom, that links back to the top of the page. For this to work I had to add a JQuery script link to the top of the page, although I have read that the loading speed may be better if I put it at the bottom. I am going to investigate this more!
